# #1819 序列中不同最大公约数的数目

  • 难度：困难
  • 合理使用筛法加速枚举
  • 原题链接

注意点

  • 不要直接枚举数组里的数，一定会超时，换个方向考虑，直接枚举最大公因数，把枚举的范围缩小到 [1,maxelement][1,max_element][1,maxe​lement]。
  • 挨个计算 nums[i]nums[i]nums[i] 是否能被 gcd 整除也会超时，再换个方向考虑，用查表的方式存下 [1,maxelement][1,max_element][1,maxe​lement] 中的数是否存在在数组中，这样只需要 O(maxelement)∗O(1)O(max_element)*O(1)O(maxe​lement)∗O(1) 的复杂度。
  • 不要遍历比当前 gcd 小的数，缩短时间。
