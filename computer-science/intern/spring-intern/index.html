# 写在前面

随便写写，觉得一个阶段告一段落需要有点总结性文字纪念一下。由于写得不那么及时，可能很多记忆都模糊了。

一些基本情况

进入暑期实习的时候攒了 3 段实习，简历上写了 2 个项目（太多了删掉一点，把主角留给实习），一些无关紧要的奖学金和荣誉。反正把舞台交给实习和项目就对了。

# 日常实习

从 2 月初开始投的，因为来不及复习开发八股，所以投的都以测试和产品为主。

  • 小红书（测试开发实习生）：2023/02/09 捞简历；2023/02/14 一面；2023/02/15 二面；2023/02/15 当天下午发 offer。
  • 美团（隐私计算项目实习）：2023/02/07 捞简历；2023/02/14 一面；一面挂。
  • 蔚来（测试开发实习生）：2023/02/07 捞简历；2023/02/08 一面 + 二面；当天下午 oc 之后拒 offer。

# 暑期实习

因为可以在小红书一直实习下去，所以对于找暑期实习并不是很迫切。而且在遭受了互联网毒打之后，对百度、阿里、腾讯、字节、美团之类的公司都不是很热衷，既然不想在这样的公司长期呆下去，就不必浪费时间继续再换个地方工作几个月。对暑期有留用的公司投递得比较谨慎，基本上分为几类：

  1. 真的很想去的（都是外企）：
    • PayPal（开发）：2023/03/28 投递；2023/03/29 笔试；笔试之后无下文。
    • 野村证券（开发）：2023/03/15 投递；2023/03/24 笔试；笔试之后无下文。
    • SAP（开发）：2023/03/04 投递；2023/03/20 下午 hr 电面；2023/03/20 晚上笔试；2023/04/06 线下群面；2023/04/11 收到 oc。
  2. 以上去不了但可以凑合的（金融类且有规模的公司）：
    • 招商银行信用卡中心（开发）：2023/03/15 投递；2023/03/25 笔试；笔试之后无下文。
    • 众安保险（测试）：2023/03/17 投递；2023/03/27 笔试；2023/03/30 技术面；2023/04/06hr 面；2023/04/21 感谢信。
  3. 练手的：
    • 京东（测试）：2023/03/28 投递；2023/03/29 测评；2023/03/30 一面；2023/04/03hr 面；泡池子之后 2023/04/28 捞起来再面，拒了。
    • 华为（软件）：2023/04/04 投递；2023/04/19 笔试，拒了。
    • 美团（测试）：2023/03/30 投递；2023/04/04 笔试；2023/04/11 约面，拒了。
    • 香农投资（C++ 开发）：2023/03/15 投递；2023/03/22 一面；2023/03/24hr 面，oc 拒了。

可以看出一些规律 = 传统大厂不卷开发，非传统大厂去浑水摸鱼。

# 一些面经

其实基本上都录了音，但有点羞耻不太想听了啊啊啊啊啊…

# 小红书

  1. 主要问项目：很笼统的问题，项目做了什么，难点是什么，收获是什么。
  2. 一些测试基础：等价类划分、黑盒白盒测试、测试工具。
  3. 八股：C++ 的继承和多态、static 的各种知识、this 的作用、数据库的索引、服务器的服务部署。
  4. 手撕算法题：二叉树节点求和，两数之和。
  5. 情景题：如何测试一个登录界面。

观感：

一面是温柔同事姐姐，二面是 leader，很领世面的样子。两面都采访了同事在小红书的工作感想，还没进去就预感到会很卷，会是 tough job，不过我也坚持下来了。

# 蔚来

  1. 项目：基本上没怎么问吧，反而问我研究生怎么没拿过奖学金，我：噢拿过的没写上去，无关紧要。
  2. 测试基础：基本上没怎么问。
  3. 八股：简单的计网和数据库。
  4. 手撕算法题：判断回文串、冒泡排序。
  5. 情景题：测试抖音分享页、如果测试机有限怎么和同时协调（大无语）。

观感：

一天速通 offer，应该比较缺人，但感觉面试官不太专业。一行代码都没写，开口闭口就是这个逻辑可能实现不出来 / 写出来会有 bug。很多 “开动脑筋再想想” 最后的结果都很莫名其妙，比如判断回文串可以先把字符串翻转然后比较（？搞毛，小丑竟是我自己是吧）。面试官感觉不知道在面点什么，也没有想好什么问题，特别是二面的面试官，讲话全程慢吞吞的，没有反馈感，如果没准备好面试可以不用把进度推那么快的。而且作为 base 在上海的业务组，对上海的 985 也没有基本的了解，简直让人怀疑组里根本没有多少 985 的人（不是凡尔赛也不是自视甚高，就是觉得业务组可能缺乏行业认知）。最后就是听起来加班挺狠的，一二面都提到了工作时间的问题，虽然菜，但还不想这么早就去干苦力。以后估计也不会投了。

# 美团

  1. 主要围绕着毕设的隐私保护项目问。用到了同态加密的知识，感觉他们很缺这方面的人，想引进同态的技术，各种问我觉得 xxx 技术的效率如何，用在金融平台上可行性如何，但又不置可否。
  2. 问了一些开源代码之类的问题，问我对隐语之类的技术的了解，我：不太了解。

观感：

属于是他们没看上我，我也没看上他们，虽然不想说得很难听，但看起来真的很像套我方案的，面试体验很不好。

# 京东

因为没有开摄像头，及时记下了各种知识点。

  1. 八股：
    TCP/IP 有几层，应用层有哪些协议，http 的特点，http 请求头，http 长连接，cookie 和 session 的区别，tcp 和 udp 的区别。
  2. 实习：
    用 python 写过脚本吗，测试实习的工作主要有哪些，写自动化用例是怎么写的，用过哪些测试工具，项目交付的整体流程有哪些，app 测试有没有用过性能分析的工具，抓包用的什么工具。小红书和京东各有什么吸引你的地方。

观感：

所有的实习面试里八股问的最多的了。但本来也是投着玩，一面面试官美其名曰 “想让实习生尽早体验 618 氛围”，实际上就是希望尽快到岗缺韭菜加班了。表现得不是很想去的样子，就把我泡池子里去了（不过我本来想的也是要是收到 oc 就美美拒掉）。

# 众安

  1. 项目：
    微信小程序项目主要干了什么，自己负责哪部分，心率检测怎么做的。毕业设计项目主要做的是什么（应该是完全不懂隐私保护），有什么具体应用。
  2. 实习：
    对职业生涯是怎么规划的，系统测试、自动化测试、测试开发有什么不同。为什么不在小红书继续实习，对于职业的行业选择有什么偏向性吗。
  3. 八股：
    http 有哪些请求方式，post 和 get 有什么不同。

观感：

听起来也是要加班的，虽然给的钱还不错，分配的事业群也是能赚钱的，但是感觉技术真的很水（…），而且又表现得不是很想去，看起来很像骑驴找马的（实际上也确实是），又泡池子然后 gg 了。

# SAP

  1. 算法题：
    笔试两道算法题直接爆零，只记得第一道题是约瑟夫问题，第二道题一点都没做。线下群面给了两道简单算法手写，一道是判断邮箱格式是否合法，一道是两数之和。
  2. 其他面试题：
    无领导小组讨论题目是销毁过期面包遇到难民和记者，如何让大家都满意。技术面问了 web 3.0 的知识（大家都不太了解，我凭着专业硬答了一些）；如何在 618 大促来临之际增加电商服务稳定性，大家头脑风暴，主要还是高并发；http 状态码有哪些（有个大哥从 1XX 背到了 5XX），cookie 和 JWT token 的区别。
  3. hr 电面：
    英文自我介绍，问了一些实习的工作，家乡哪里的，毕业之后职业规划如何，为什么不想呆在小红书，为什么想来 SAP，对于学习新编程语言的看法如何。

观感：

总体过下来是最顺，发挥也是最好的吧，虽然笔试考崩了痛哭了一晚上，但感谢自己痛定思痛，好好准备了剩下的面试。大三面 SAP 的时候感觉 HR 非常 mean，听说我要继续读研把我怼了一顿，这次碰到的倒是都挺正常。

# 写在最后

想了一想，自己这个春招水平可能只能算比上不足、比下有余，如果是大佬点进来可能是见笑了。如果非要总结点什么的话，那么我觉得有几点是自己可以继续坚持的，写下来让自己时常看看。

  1. 看清楚自己站在哪，又能迈多大步子。明确目标，缩小范围，劲用在刀刃上。想要的就努力得到，不想要的毋论有无都不必伤心。（为什么突然想到了上周组会讲的 refined search，有海投的心，但还得小范围才能有高效的结果。）
  2. 可以接受短期性吃苦，不接受无期的苦熬。不用身体换工资，我需要有自己的生活。
  3. 路还很长，继续走着继续看～
