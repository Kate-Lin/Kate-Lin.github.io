# 密钥协商

# 密钥预分配

可信权威机构（Trust Authority, TA）以一种安全的方式为网络中的每个用户 “提前” 分发密钥信息，分发的时候需要一个安全信道。每对用户可以根据掌握的密钥信息来确定一个密钥，该密钥只有他们两人知道。

# 会话密钥分配

当网络用户请求会话密钥时，一个在线的 TA 会选择会话密钥并通过一个交互协议分发给他们。
会话密钥用于在指定的、相当短的时间内加密信息。

# 密钥协商

网络用户通过一个交互协议来建立会话密钥。密钥协商可以是基于对称密码体制的，也可以是基于公钥密码体制的，通常不需要一个在线的 TA。
