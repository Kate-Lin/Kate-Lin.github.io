# LowMC 简介和主要名词解释

# LowMC

LowMC 是 SPN 分组密码族中较为新颖的一类，目前被用于 NIST 后量子密码征集第三轮备选签名方案 Picnic 的底层。LowMC 旨在实现低乘法复杂度，并且实例化较为灵活，用户可以独立选择 S 盒数量、线性层、密钥调度函数等参数，进行 LowMC 的实例化。低乘法复杂度的实现方式是使用部分 S 盒层和随机密集线性层。所谓部分 S 盒层，就是只有部分状态位通过 S 盒，其余的状态位都应用了特征映射（identity mapping）。

LowMC 自发布以来，已经进行了两轮更新完善。在针对 LowMC 的高阶差分攻击和插值攻击提出后，LowMC v2 被提出，可用于抵抗上述攻击，新的公式可以用于确定安全的轮数数量。针对 LowMC v2 的特性，一种新的攻击方式被提出，即差分枚举攻击。为抵抗差分枚举攻击，LowMC 也进入了 v3 时代。本文提到的 LowMC 指的都是 LowMC v3。

# LowMC-M：

LowMC-M 是 LowMC 密码方案的一种变体，是一种可调分组密码（tweakable block cipher）。LowMC-M 在 LowMC 中嵌入了后门，在每轮轮函数的最后加入了新函数 AddSubTweak。虽然敌手可能会获得可调的能力，但 LowMC-M 的参数在此情况下也还是安全的。

# 差分枚举攻击（Difference Enumeration Attack）

差分枚举攻击是一种选择明文攻击。主要思路如下：攻击者给出一个输入差分（一对明文的差分），根据标准差分密码分析中的差分传播规则，生成所有可达的输出差分的列表。如果生成的可达列表比穷举的大小小很多，那么就可以利用这种不均匀性，以比穷举更高的效率恢复出正确的输出差异。在实践中，往往采用 “meet-in-the-middle” 的方式实现该攻击。

# 代数方法（Algebraic Techniques）

在 SPN 攻击的相关工作中，对本文而言较为有借鉴意义的 Bar-On 提出的，用代数方法恢复 SPN 部分非线性层的差分轨迹的方法，发表于 EUROCRYPT 2015。Bar-On 在方案中引入了中间变量表示第一轮 S 盒层的输出，并控制激活的 S 盒数量在适中的范围内。然后，在活跃 S 盒数量低于预设值的限定条件下，遍历所有可能的差分模式。对于每种模式，活跃 S 盒的输出赋中间变量，非活跃 S 盒的输出可根据 “输入、输出差分等于零” 的特征，建立线性表达式表示。最后可以通过解方程组的方式还原出差分轨迹，进一步还原出密钥。

上述的方法就是典型的代数攻击方法。代数方法首先构造多元方程组描述目标问题，然后求解方程组。在 Bar-On 的方案中，未知数为每轮引入的中间变量，方程组为非活跃 S 盒蕴含的线性关系。如果给出的方程组本身是线性的，那么可以通过高斯消去进行求解。如果是非线性的方程组，求解是 NP-hard 问题，但可以根据分组密码内的一些冗余代数性质简化求解，或可以使用 “guess-and-determine” 的方法简化求解过程。
