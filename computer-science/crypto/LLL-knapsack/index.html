# 用 LLL 攻击背包密码

1982 年，Shamir 使用 Lenstra 的方案攻破了 Merkle-Hellman 背包方案，使用了 LLL 算法作为子程序。（A polynomial time algorithm for breaking the basic Merkle-Hellman cryptosystem）

攻击的是 single-iteration Merkle-Hellman cryptosystem，是 MH 背包类密码中最简单的一种。（未提供 multi-iteration 的攻击方法）

# how to attack

最重要的思想是如何将 LLL 规约短向量的作用应用到背包密码中。

首先我们看到背包密码加密的格式（以矩阵形式表示）：

s=Enca(x)=⟨a,x⟩=a1×n⋅xn×1s=Enc_a(x)=\langle\boldsymbol{a},\boldsymbol{x}\rangle=\boldsymbol{a}_{1\times n}\cdot \boldsymbol{x}_{n\times 1}s=Enca​(x)=⟨a,x⟩=a1×n​⋅xn×1​

其中a\boldsymbol{a}a 为公钥，x\boldsymbol{x}x 是攻击者需要获得的信息，sss 为密文。由背包的定义可知，x\boldsymbol{x}x 矩阵的每行都是 0/1，这种特殊的结构给了攻击利用的空间。

如何将这个加密的结构和格结合起来呢？这里需要引入一个重要的等价等式：

BK=Y\boldsymbol{BK}=\boldsymbol{Y}BK=Y

其中B\boldsymbol{B}B 是一个(n+1)×(n+1)(n+1)\times(n+1)(n+1)×(n+1) 维的矩阵，结构如下：

B=(In×n0n×1a1×n−s1×1)B=\left( \begin{matrix}I_{n\times n}&0_{n\times 1}\\\boldsymbol{a}_{1\times n}&-s_{1\times 1}\end{matrix}\right)B=(In×n​a1×n​​0n×1​−s1×1​​)

K\boldsymbol{K}K 是一个(n+1)×1(n+1)\times 1(n+1)×1 维的矩阵，结构如下：

BK=Y\boldsymbol{BK}=\boldsymbol{Y}BK=Y 和ax=s\boldsymbol{ax}=\boldsymbol{s}ax=s 是等价的。因为根据SK=Y\boldsymbol{SK}=\boldsymbol{Y}SK=Y 的结构展开可以得到两个等式：x=xx=xx=x 和ax−s=0ax-s=0ax−s=0，即ax=sax=sax=s。所以，若BK=YBK=YBK=Y 中的KKK 或YYY 可解，那么xxx 就可解，明文就能被还原。LLL 算法的输入是一组格基，输出是若干正交短向量的格基。通过引入BK=Y\boldsymbol{BK}=\boldsymbol{Y}BK=Y 这个等式，使得 LLL 算法的输入和输出都有了对应的变量。

首先思考如何匹配 LLL 算法的输入：B 矩阵的各分块矩阵都是已知的，考虑将 B 矩阵的各列(b0,...,bn)(b_0,...,b_n)(b0​,...,bn​) 构成一个格

再考虑匹配 LLL 算法的输出：向量 Y 有两个特殊的点：
