# 2023/01/19

# 最近修改：

  1. 把配色大换血，主体改为蓝绿色调。
  2. 更新了搜索引擎的索引收录，目前已通过了百度盒谷歌的站点验证，应该过段时间就可以搜到本站的内容了。
  3. 增加了两个新的菜单栏：
    1. 文章-标签 ：目前已经更新了 20 + 标签
    2. 友情链接 ：虽然现在也没有人来和我投稿，糊逼博客一枚，我链接我自己。
  4. 为配合搜索引擎收录，还是绑定了（二级）域名，现在可以通过 https://blog.bullet-holes.cn 访问，没有切主域名是因为备案会很麻烦，github.io 没法用 ip 备案（万一 ip 变动），会牵涉到之前的备案，很麻烦，同时 https://www.bullet-holes.cn 仍可用作旧版 Ghost 博客的入口。

# 未来 todo：

  1. 把加载动画换了，学习一下怎么做的（发现原主题好像也用的是开源的，目前已经找到了更换加载动画的方法，见👉again 又是这位反转天平大佬）。
  2. 把赞赏按钮改成分享到微信的按钮（对微信分享有那么点执念的我本人），要动的是 \themes\shoka\layout\_partials\post\reward.njk （初步看了一下，能看懂，应该能下笔改）。但不知道现在微信开放的接口怎么用，应该可以调个 js，但怎么设置分享缩略图也是需要研究一下的，转发给朋友和转发至朋友圈是不是也不太一样？

# 2023/01/17

# 最近修改：

  1. 把 gravatar 和 cdn.jsdelivr 做了反代，在墙内可以正确加载评论区头像和评论表情。
  2. 因为 jsdelivr 不是实时更新，踩了个大坑（一直调用到没更新的静态文件，反复踩坑），最后给资源打了版本才解决的。
